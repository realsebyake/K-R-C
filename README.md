# K&R C

The purpose of this repo is to track exercises and examples covered in "The C Programming Language" by K&R. Repo is intended to be kept for personal records and archive.

Note that this can also serve well as a resource to solve the exercises in the book (alternatively to buying the exercises book)

Also some programs will be using void functions instead of just int mains primarily because I am integrating K&R exercises or examples into one singular file so I dont add a billion other files for just one sub-chapter.
